This plugin replaces the default more tag with a sliding jQuery hidden content functionality.

== Description ==

This plugin replaces the default more tag with a sliding jQuery hidden content functionality.  Instead of hiding the full post and making the visitor go to the single post page this will allow them to expand the content on the existing page with a smooth jQuery action.

== Installation ==

   1. Install the plugin either via FTP or Admin backend
   2. Activate the plugin
   3. Edit text settings for plugin.
